# MusicOptions 
Additional configuration options for @minecraft/server.World.playMusic/@minecraft/server.World.queueMusic methods.

## Properties

- fade
fade?: number;

Specifies a fade overlap for music at the end of play.

Type: number

- loop
loop?: boolean;

If set to true, this music track will play repeatedly.

Type: boolean

- volume
volume?: number;

Relative volume level of the music.

Type: number
