# StructureMirrorAxis 
Specifies how a structure should be mirrored when placed.

## Constants

- None
None = "None"

No mirroring.

- X
X = "X"

Structure is mirrored across the X axis.

- XZ
XZ = "XZ"

Structure is mirrored across both the X and Z axes.

- Z
Z = "Z"

Structure is mirrored across the Z axis.
